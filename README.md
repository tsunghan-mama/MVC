# MVC Image Cloning
---

## Quick Start

### Installation

```bash
git clone https://github.com/tsunghan-mama/MVC.git
cd MVC
conda env create -f environment.yml
```

### Run the App

```bash
flask run --host=<ip> --port=<desired-port>
```

Then, open the browser and enjoy it on http://\<ip\>:\<port\>.
